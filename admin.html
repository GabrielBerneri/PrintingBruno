Guía rápida del panel

El panel separa cada decisión importante para que no se mezcle lo comercial con lo operativo. Las vistas generales muestran un resumen simple; el detalle de cada pedido te deja ver y editar cada dimensión por separado.

Dashboard

Te muestra el estado general del negocio: ventas cobradas, cobros pendientes, stock bajo y últimos movimientos.

Pedidos

Administrás el cobro y la operación. El cobro dice si está pago; la operación dice en qué etapa va el trabajo.

Productos

Gestionás catálogo, imágenes y stock físico. El stock reservado se calcula aparte cuando entra un pedido.

Correos

Revisás el historial de emails enviados o fallidos. Sirve para control y soporte, no para cambiar estados.

Cómo leer pedidos Las listas te muestran un estado resumido. El detalle separa cobro y operación.

Usá el resumen para entender rápido qué está pasando. Entrá al detalle cuando necesites cambiar el cobro o avanzar el trabajo.

Cobro

Indica si el pago está pendiente, en revisión, aprobado, rechazado, cancelado o reembolsado.

Operación

Indica la etapa del trabajo: en cola, en producción, listo, enviado o entregado.

Ejemplos
  • Pendiente de pago: todavía no hay cobro confirmado.
  • Pagado · En producción: ya se cobró y el pedido se está haciendo.
  • Cancelado: el pedido no sigue activo.

Pedidos

Cargando pedidos... Usá la cola para entrar por tarea y el detalle para confirmar cobros o avanzar producción.
